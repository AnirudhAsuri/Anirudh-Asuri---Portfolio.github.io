Context Steering in 3D

Context Steering is a navigation AI technique originally developed for top down 2D environments.
This project took this concept and adapted it to a 3D environment.
Generally, people tend to use prebuilt nav-mesh agents for their AI navigation. However, in Unity, using this prevents us from using our own custom movement systems.
This particular problem is what this project hopes to fix.
Context Steering is a lightweight system that uses raycasts to get 2 maps of data regarding the target position and obstacles in the way.
This data is then used to make decisions in real-time.
This project contains a playable demo that uses a custom acceleration based movement system, thus using Context Steering in 3D for the AI Navigation.

View the project here - Context Steering/Ponchon Demo Itch.io page.(An open source code base is in development)
